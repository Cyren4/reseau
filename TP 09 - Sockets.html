Marquer la question

Texte informatif

  • Lire le document compagnon du TP09
  • Concevoir les deux programmes python suivants :

    • Exécuter le serveur et le client sur votre machine locale en utilisant l'adresse IP de loopback 
    • Le client est supposé connaître le numéro de port du serveur, à savoir 65432
    • Le serveur doit :
      • importer le module socket
      • créer une extrémité de connexion TCP
      • attacher cette extrémité à l'adresse IP loopback et au port d'écoute 65432
      • se mettre en attente de connexion de la part du client
      • une fois le client connecté, lui envoyer le message "Bonjour Client, je suis le serveur 65432, la connexion est établie" 
      • recevoir un message du client et l'imprimer
      • recevoir un second message du client et l'imprimer 
      • fermer la connexion 
    • Le client doit :
      • importer le module socket
      • créer une extrémité de connexion TCP
      • attacher cette extrémité à l'adresse IP loopback
      • se connecter au port 65432 du serveur 
      • recevoir un message de la part du serveur et l'imprimer
      • envoyer le message "Bonjour Serveur 65432, nous sommes donc dans la phase de transfert" au serveur 
      • envoyer le message "Serveur 65432, je vous dis au revoir"
      • fermer la connexion
  • Exécuter ces deux programmes dans deux terminaux différents
